Quando estiver pronto para integração real com Firebase:

1. Instale dependências:
   cd backend
   npm install

2. Crie projeto em Firebase Console

3. Gere chave de serviço (Service Account JSON)

4. Configure backend/.env com as credenciais

5. Substitua backend/index.js com versão Firebase

6. Reinicie servidor
